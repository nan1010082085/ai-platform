# 项目熟悉指南

## 概述

本目录包含帮助专家 Agent 熟悉项目的所有资料。通过系统地学习这些资料，Agent 可以快速掌握项目的架构、代码和开发流程。

## 学习路径

### 第一阶段：快速了解 (30 分钟)

1. **阅读快速入门指南** (`quick-start.md`)
   - 了解项目是什么
   - 理解核心架构
   - 掌握基本操作

2. **查看项目架构图** (`architecture.md`)
   - 理解整体架构
   - 了解模块关系
   - 掌握数据流

### 第二阶段：深入学习 (2 小时)

1. **阅读项目概览** (`project-overview.md`)
   - 了解项目定位
   - 熟悉技术栈
   - 掌握核心功能

2. **按照探索清单学习** (`explore-project.md`)
   - 系统地探索代码
   - 记录学习笔记
   - 标记完成任务

### 第三阶段：动手实践 (4 小时)

1. **启动项目**
   ```bash
   cd app && pnpm install && pnpm dev
   ```

2. **阅读核心代码**
   - 主 Store (`app/src/stores/ai.ts`)
   - 主对话页面 (`app/src/views/AiChatView.vue`)
   - 通信桥 (`app/src/utils/bridge.ts`)

3. **运行测试**
   ```bash
   cd app && pnpm test
   ```

4. **尝试修改**
   - 修改一个简单的 UI 文案
   - 添加一个 console.log
   - 观察变化

## 常见问题

### Q: 学习这个项目需要什么基础？

A: 需要掌握：
- Vue 3 + TypeScript
- Pinia 状态管理
- Vite 构建工具
- 基本的 HTTP/WebSocket 知识

### Q: 学习这个项目需要多长时间？

A: 取决于基础和投入时间：
- 快速了解：1-2 天
- 深入学习：1 周
- 熟练掌握：2-4 周

### Q: 学习过程中遇到问题怎么办？

A: 可以：
1. 查阅官方文档
2. 搜索相关资料
3. 请教同事
4. 记录问题，稍后解决

### Q: 如何检验学习效果？

A: 可以：
1. 完成检查点任务
2. 尝试解决实际问题
3. 参与代码审查
4. 贡献代码
