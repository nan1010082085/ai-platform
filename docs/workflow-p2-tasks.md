## 任务分解

### N-3: 插件在线编辑（5d）

**目标**：在 PluginCenterView 中添加在线编辑功能

**子任务**：
1. 后端：添加插件配置 CRUD API
2. 前端：添加编辑对话框组件
3. 前端：添加 JSON 编辑器
4. 集成：保存到本地配置文件

### O-3: Skill 拼装顺序规范（1d）

**目标**：明确 Skill 拼装顺序和冲突处理规则

**子任务**：
1. 分析现有 Skill 拼装逻辑
2. 编写规范文档
3. 添加单测验证

### O-4: Skill 多语言（2d）

**目标**：支持多语言 Skill

**子任务**：
1. 扩展 Skill 类型定义（添加 locale 字段）
2. 修改 Skill 解析逻辑（按 locale 匹配）
3. 添加多语言 Skill 示例

### O-7: MCP 租户隔离 UI（3d）

**目标**：在插件中心添加租户级 MCP 配置管理

**子任务**：
1. 后端：扩展 MCP 配置模型（添加 tenantId）
2. 前端：添加租户选择器
3. 前端：按租户过滤 MCP 列表

### O-8: Chat 空状态引导词配置化（2d）

**目标**：将 Chat 空状态引导词从硬编码改为配置化

**子任务**：
1. 后端：添加引导词配置 API
2. 前端：修改 AiChatPanel 读取配置
3. 前端：添加引导词设置 UI

### O-11: Pack spec v1 + 签名（3d）

**目标**：定义插件包规范 v1 并支持签名

**子任务**：
1. 定义 Pack 规范（manifest.json 结构）
2. 实现打包脚本
3. 实现签名验证

### O-12: 插件级 metrics（3d）

**目标**：为插件添加性能指标收集

**子任务**：
1. 定义插件指标类型
2. 实现指标收集器
3. 在监控页面展示插件指标
