# Plugin Scaffold

Minimal Expert plugin package for Schema Platform AI.

## Files

- `expert.json` — Expert definition (import via Plugin Center)

## Usage

1. Edit `expert.json` (id, prompts, tools)
2. Open AI app → Plugin Center → Import / paste JSON
3. Enable the expert and select it in chat

See [third-party-plugin-guide.md](../third-party-plugin-guide.md) for Tool / Skill / MCP details.
