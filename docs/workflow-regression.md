## 回归测试执行

### 执行顺序

1. **启动服务**
   ```bash
   cd server && pnpm dev
   cd ai/app && pnpm dev
   ```

2. **运行单元测试**
   ```bash
   cd server && pnpm test
   cd ai/app && pnpm test
   ```

3. **运行配置校验**
   ```bash
   cd server && pnpm validate:tools
   ```

4. **手动 UI 测试**
   - 按 UI 回归清单逐项测试

5. **API 接口测试**
   - 使用 curl 或 Postman 测试

6. **文档检查**
   - 检查文件存在性和内容

### 通过标准

- 所有 UI 组件正常加载
- 所有 API 接口返回正确
- 所有功能按预期工作
- 所有文档完整准确
- 无控制台错误
- 无 TypeScript 类型错误
